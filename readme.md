# npm-v

> Get installed npm version


Yup it only do that :tada:


## Install

```bash
npm install --save npm-v
```


## Usage

```js
import npmVersion from 'npm-v'

console.log(npmVersion)
// {
//   version: "3.10.8",
//   major: 3,
//   minor: 10,
//   patch: 8
// }
```
